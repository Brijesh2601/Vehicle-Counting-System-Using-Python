# Vehicle Counter Project - Setup Guide

## 1. Install Dependencies
Open your terminal (Command Prompt) in this folder and run:

pip install -r requirements.txt

## 2. Run the App
Run the following command to start the dashboard:

streamlit run app.py

## Note about 'libs' folder
This project contains a 'libs' folder with some pre-installed libraries. 
Do not delete it, as the application uses it for stability.
